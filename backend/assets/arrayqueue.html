Understanding ArrayQueue in C++

What is an ArrayQueue?

An ArrayQueue is a linear data structure that follows the First In, First Out (FIFO) principle. It is implemented using an array and provides efficient enqueue and dequeue operations.

Characteristics of ArrayQueue

  • Fixed Size: The size of the queue is fixed and must be defined at the time of creation.
  • FIFO Principle: The first element added to the queue is the first one to be removed.
  • Efficient Operations: Enqueue and dequeue operations have a time complexity of O(1).

Implementing an ArrayQueue

Defining the Queue

class ArrayQueue {
private:
    int *arr;
    int front;
    int rear;
    int capacity;
    int count;

public:
    ArrayQueue(int size);
    ~ArrayQueue();
    void enqueue(int x);
    int dequeue();
    int peekFront();
    int peekRear();
    bool isEmpty();
    bool isFull();
};

Different Operations

Constructor

ArrayQueue::ArrayQueue(int size) {
    arr = new int[size];
    capacity = size;
    front = 0;
    rear = -1;
    count = 0;
}

Destructor

ArrayQueue::~ArrayQueue() {
    delete[] arr;
}

Enqueue Operation

void ArrayQueue::enqueue(int x) {
    if (isFull()) {
        std::cout << "Overflow\n";
        return;
    }
    arr[++rear] = x;
    count++;
}

Dequeue Operation

int ArrayQueue::dequeue() {
    if (isEmpty()) {
        std::cout << "Underflow\n";
    }
    int x = arr[front];
    for (int i = 0; i < rear; i++) {
        arr[i] = arr[i + 1];
    }
    rear--;
    count--;
    return x;
}

Peek Front Operation

int ArrayQueue::peekFront() {
    if (isEmpty()) {
        std::cout << "Underflow\n";
    }
    return arr[front];
}

Peek Rear Operation

int ArrayQueue::peekRear() {
    if (isEmpty()) {
        std::cout << "Underflow\n";
    }
    return arr[rear];
}

Check if Empty

bool ArrayQueue::isEmpty() {
    return count == 0;
}

Check if Full

bool ArrayQueue::isFull() {
    return count == capacity;
}

Conclusion

ArrayQueue is a fundamental data structure that is widely used in many applications. Understanding its implementation and operations is crucial for any programmer.
