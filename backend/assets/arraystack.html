Understanding ArrayStack in C++

What is an ArrayStack?

An ArrayStack is a linear data structure that follows the Last In, First Out (LIFO) principle. It is implemented using an array and provides efficient push and pop operations.

Characteristics of ArrayStack

  • Fixed Size: The size of the stack is fixed and must be defined at the time of creation.
  • LIFO Principle: The last element added to the stack is the first one to be removed.
  • Efficient Operations: Push and pop operations have a time complexity of O(1).

Implementing an ArrayStack

Defining the Stack

class ArrayStack {
private:
    int *arr;
    int top;
    int capacity;

public:
    ArrayStack(int size);
    ~ArrayStack();
    void push(int x);
    int pop();
    int peek();
    bool isEmpty();
    bool isFull();
};

Different Operations

Constructor

ArrayStack::ArrayStack(int size) {
    arr = new int[size];
    capacity = size;
    top = -1;
}

Destructor

ArrayStack::~ArrayStack() {
    delete[] arr;
}

Push Operation

void ArrayStack::push(int x) {
    if (isFull()) {
        std::cout << "Overflow\n";
        return;
    }
    arr[++top] = x;
}

Pop Operation

int ArrayStack::pop() {
    if (isEmpty()) {
        std::cout << "Underflow\n";
        exit(EXIT_FAILURE);
    }
    return arr[top--];
}

Peek Operation

int ArrayStack::peek() {
    if (!isEmpty()) {
        return arr[top];
    } else {
        exit(EXIT_FAILURE);
    }
}

Check if Empty

bool ArrayStack::isEmpty() {
    return top == -1;
}

Check if Full

bool ArrayStack::isFull() {
    return top == capacity - 1;
}

Conclusion

ArrayStack is a fundamental data structure that is widely used in many applications. Understanding its implementation and operations is crucial for any programmer.
