Understanding Heap Sort in C++

What is Heap Sort?

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It is efficient and has a good time complexity, making it suitable for large datasets.

Characteristics of Heap Sort

  • Binary Heap: Utilizes a binary heap structure, either max-heap or min-heap.
  • In-place: Requires only a constant amount of additional space.
  • Not Stable: The relative order of equal elements may not be preserved.

Time Complexities

The time complexities of the various operations in Heap Sort are as follows:

  • Building the Heap: This takes O(n) time. Each insertion operation in a heap takes O(log n) time, but building the heap from an unordered array takes linear time.
  • Heapify Operation: The heapify operation, which is called O(n) times, takes O(log n) time for each call.
  • Sorting the Array: Extracting the maximum element from the heap and heapifying the remaining elements n times takes O(n log n) time.

Overall Time Complexity: The overall time complexity of Heap Sort is O(n log n) for all cases (best, average, and worst) because it involves heapifying the elements and then extracting elements from the heap.

Implementing Heap Sort

Defining heapify function

Heapify is a function that transforms a binary tree into a heap, which is a special kind of binary tree that satisfies the heap properties.

void heapify(int arr[], int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest])
        largest = left;

    if (right < n && arr[right] > arr[largest])
        largest = right;

    if (largest != i) {
        swap(arr[i], arr[largest]);
        heapify(arr, n, largest);
    }
}

Defining heapSort function

Heap Sort is a comparison-based sorting technique based on the binary heap data structure. It sorts an array by first building a heap from the array and then repeatedly extracting the maximum element from the heap and adjusting the heap structure accordingly.


        void heapSort(int arr[], int n) {
        for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);

        for (int i = n - 1; i > 0; i--) {
        swap(arr[0], arr[i]);
        heapify(arr, i, 0);
        }
        }

Main Function

int main() {
    int arr[] = {12, 11, 13, 5, 6, 7};
    int n = sizeof(arr)/sizeof(arr[0]);

    heapSort(arr, n);

    cout << "Sorted array is: ";
    for (int i = 0; i < n; ++i)
        cout << arr[i] << " ";
    cout << endl;
    return 0;
}
