Implementing a Stack using Linked List in C++

What is a Stack?

A stack is a linear data structure that follows the Last In First Out (LIFO) principle. The operations are performed in the following order:

  1. Push: Adds an item to the top of the stack.
  2. Pop: Removes an item from the top of the stack.

A stack can be effectively implemented using a linked list, providing dynamic size adjustment capabilities.

Stack Operations

  • Push: Add to the head of the linked list.
  • Pop: Remove from the head of the linked list.
  • Top: Return the data of the head node.
  • isEmpty: Check if the stack is empty.

Implementing a Stack

Using the Node Class

We'll use the same Node class from the linked list implementation:

class Node {
public:
    int data;
    Node *next;

    Node(int d) : data(d), next(nullptr) {}
};

Stack Class

class Stack {
private:
    Node *top;

public:
    Stack() : top(nullptr) {}
    ~Stack() { while (!isEmpty()) pop(); }

    void push(int data);
    int pop();
    int getTop();
    bool isEmpty();
};

Stack Operations

Push

void Stack::push(int data) {
    Node *newNode = new Node(data);
    newNode->next = top;
    top = newNode;
}

Pop

int Stack::pop() {
    if (isEmpty()) {
        throw std::out_of_range("Stack is empty");
    }
    int data = top->data;
    Node *temp = top;
    top = top->next;
    delete temp;
    return data;
}

Get Top

int Stack::getTop() {
    if (isEmpty()) {
        throw std::out_of_range("Stack is empty");
    }
    return top->data;
}

Is Empty

bool Stack::isEmpty() {
    return top == nullptr;
}

Benefits of Using Linked List

  • Dynamic Size: Grows and shrinks as needed.
  • Fast Operations: O(1) time for push, pop, and top.
  • Memory Efficiency: Allocates only what's required.

Common Stack Applications

  • Function Call Management: In program execution.
  • Expression Evaluation: Infix to postfix conversion.
  • Backtracking: In algorithms like maze solving.
  • Undo Functionality: In text editors.

Example Usage

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);

    cout << s.getTop() << endl;  // Outputs: 30
    cout << s.pop() << endl;    // Outputs: 30
    cout << s.getTop() << endl;  // Outputs: 20

    s.push(40);
    while (!s.isEmpty()) {
        cout << s.pop() << " ";  // Outputs: 40 20 10
    }
    cout << endl;

    return 0;
}

Balancing Parentheses Example

bool isBalanced(string expr) {
    Stack s;
    for (char ch : expr) {
        if (ch == '(' || ch == '{' || ch == '[') {
            s.push(ch);
        } else if (ch == ')' || ch == '}' || ch == ']') {
            if (s.isEmpty()) return false;
            char top = s.pop();
            if ((ch == ')' && top != '(') || 
                (ch == '}' && top != '{') || 
                (ch == ']' && top != '['))
                return false;
        }
    }
    return s.isEmpty();
}

Conclusion

A linked list-based stack offers a flexible and efficient solution for LIFO operations. Its dynamic nature makes it suitable for a wide range of applications, particularly those with unpredictable stack size requirements.
