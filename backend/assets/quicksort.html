Understanding Quick Sort in C++

What is Quick Sort?

Quick Sort is a highly efficient sorting algorithm and is based on partitioning an array into smaller sub-arrays. It follows the divide and conquer approach.

Characteristics of Quick Sort

  • Divide and Conquer: The array is divided into sub-arrays around a pivot element.
  • Efficient for Large Data Sets: Quick Sort is generally faster than other O(N log N) algorithms for large data sets.
  • In-Place Sorting: Does not require extra space, sorts the array in place.

Time Complexity

  • Best Case: O(N log N)
  • Worst Case: O(N2)
  • Average Case: O(N log N)

Implementing Quick Sort

Different Operations

Swap Elements

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

Partition Function

int partition(int arr[], int low, int high) {
    int pivot = arr[low];
    int i = low, j = high;
    while (i < j) {
        while (arr[j] > pivot)
            j--;
        while (arr[i] < pivot)
            i++;
        swap(&arr[i], &arr[j]);
    }
    return j;
}

Quick Sort Function

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

Print Array

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

Main Function

int main() {
    srand(time(NULL));

    int r, n = 10;
    
    int *arr = new int[n];

    for (int i = 0; i < n; i++) {
        r = rand();
        arr[i] = r;
    }
    cout << "\nUnsorted array: ";
    printArray(arr, n);

    quickSort(arr, 0, n - 1);

    cout << "\nSorted array: ";
    printArray(arr, n);
    
    return 0;
}
